        • Light mode
          Off = warm dark (default). On = warm cream. Your choice persists across reloads.

        • Manage uploads
          View on-disk size of phone uploads per project and delete them.
        • Read messages aloud
          When on, tapping any word in an assistant message starts reading from there with the chosen voice. Off = no speak buttons, no audio.

          Pick which system voice reads assistant messages aloud. iOS "Enhanced" voices sound the most human.
